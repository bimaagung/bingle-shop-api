Server kamu minimal memiliki API dengan fungsi berikut:
1. Register pelanggan baru
2. Login pelanggan
3. Menampilkan data item
4. Membuat pesanan baru
5. Memperbarui status pesanan

Criteria :
1. Rancangan database menggunakan ERD sesuai studi kasus (1 point).
2. Server REST API dibuat dengan Express (2 point).
3. Penyimpanan data ke PostgreSQL dilakukan dengan Sequelize (2 point).
4. Terdapat API register & login (2.5 point).
5. Terdapat API untuk membaca, menyimpan, memperbarui, dan menghapus data (2.5 point).
